Method Path Description
get / route to the home page
post /user route to login
get /users Route to get users
get /user/:id Get a single user
put user/:id Update a single user
put /admin/:id Toggle admin state
post /register registe a new user
delete /user/:id delete a user
get /items get all items
get /item/:id get a single item
post /items add a new item
put /item/:id update a single item
delete /item/:id delete a single item
post /user/:id/cart Add a new cart item
get /user/:id/carts Get all cart items for a single user
put /user/:uid/cart/:cid update a specified cart for a specified user
put /user/:id/cart Update all cart items for a specific user
delete /user/:id/cart Delete all cart items for a specific user
delete /user/:uid/cart/:cid Delete a single cart item for a specific user
